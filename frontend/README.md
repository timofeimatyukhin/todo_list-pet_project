# Todo List Pet Project

Full-stack TODO application with a React frontend and Spring Boot backend.

## Features

- View all tasks
- Get task by id
- Create task
- Update task title/text
- Toggle task status (`IN_PROGRESS` / `DONE`)
- Delete task

## Tech Stack

### Frontend
- React 19
- TypeScript
- Vite
- TanStack React Query

### Backend
- Spring Boot
- Spring Web MVC
- Spring Data JPA
- PostgreSQL
